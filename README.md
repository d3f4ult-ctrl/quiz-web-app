## Features
- **Welcome Screen**: Start button + rules before the quiz begins.
- **6 multiple choice questions** with instant feedback.
- **Score tracking**: Shows final score + percentage at the end.
- **Restart button**: Play again without refreshing.
- **Responsive design**: Works on phone + desktop.

## Tech Stack
- HTML5 for structure.
- CSS3 for styling + animations.
- Vanilla JavaScript build for quiz logic.

## How to run
1. Download/clone this repo.
2. Make sure that you have all files in this repo.
